LdLattice_t Class Reference

Detailed Description

Graphical lattice representation.

Lattice drawing data structure The |LdLattice| holds all data used internally by the lattice drawing algorithms.

|Node| is a list of the nodes. The nodes may appear in any order, they need not be sorted, and node 0 need not be the bottom node.
