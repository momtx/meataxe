Description

This program calculates the N-th power of a matrix or permutation. The exponent N may be negative. For compatibility with the old zsm program, the power may be specified in the zsm format. The following example shows two equivalent ways to calculate the 69-th power of a matrix: 

zpo matrix 69 result
zpo matrix pwr69 result
